MONK -- social network for A.I.

MONK is a distributed interactive machine learning framework for both users and scientists with a focus on the agility of building and sharing machine learning models. pymonk is a version implemented in python

$ git clone http://github.com/xumiao/pymonk.git
$ python setup.py install
